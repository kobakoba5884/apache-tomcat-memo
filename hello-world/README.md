# Hello World Web Application

This is a simple Java web application that demonstrates the usage of Apache Tomcat and Gradle for building and deploying the application.

## Prerequisites

- Java Development Kit (JDK) 8
- Apache Tomcat 8.5.88
- Gradle 6.9

## Building and Deploying the Application with apache and tomcat

To build and deploy the application, run the following command:

```
    chomod +x ./excute-app.sh

    ./excute-app.sh
```
